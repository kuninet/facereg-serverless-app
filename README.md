# Faces Registration Serverless App (顔写真登録アプリ)

AWSのサーバーレスアーキテクチャを活用した、低コストで運用可能な受付用の顔写真登録アプリケーションです。

## プロジェクト概要
受付に設置したキオスク端末（iPad等を想定）から、来訪者の顔写真と受付情報を登録し、管理者がクラウド上で一覧表示・ダウンロードできるシステムです。継続課金を最小限に抑えるための構成を採用しています。

## 構成
* **フロントエンド**: React/Vite 等（S3 + CloudFrontホスティング）
* **バックエンド**: AWS SAM (API Gateway + Lambda + DynamoDB)
* **ストレージ**: Amazon S3 (署名付きURLによる直接アップロードを利用)

## 環境ごとの変数の管理について
当リポジトリはパブリック公開用のため、インフラに関する固有のリソース名やカスタムドメイン名は、コード・ドキュメント内でダミー値（例: `example.com`）にマスキングされています。
実際の運用環境にデプロイする際は、以下の方法で環境固有の値を管理してください。

1. **フロントエンド環境変数**
   `.env.example` をコピーして `.env.local` を作成し、割り当てられたAPIエンドポイント等の値を設定してください。
2. **バックエンド (AWS SAM) 管理**
   SAMデプロイ時に発行される `samconfig.toml` にパラメータとしてカスタムドメインや環境プレフィックスを保存します。（`samconfig.toml`はGit管理から除外されています）
